```
usage: thalt [-l LIMIT] [-t TICKER] [-r {News-pending,LULD-pause,Merger-Effective,Regulatory-Concern,News-Released,New-Security-Offering,Corporate-Action,News-dissemination}] [-h] [--export {csv,json,xlsx}]
```

View stocks with recent trading halts [Source: NYSE]

```
optional arguments:
  -l LIMIT, --limit LIMIT
                        limit of trading halts to retrieve. (default: 20)
  -t TICKER, --ticker TICKER
                        stock ticker
  -r {News-pending,LULD-pause,Merger-Effective,Regulatory-Concern,News-Released,New-Security-Offering,Corporate-Action,News-dissemination}
  -h, --help            show this help message (default: False)
  --export {csv,json,xlsx}
                        Export raw data into csv, json, xlsx (default: )
```
